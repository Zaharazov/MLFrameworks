## Датасеты

* Diamonds Price Dataset (регрессия)  
Источник: [Kaggle](https://www.kaggle.com/code/tanshihjen/linear-regression-diamonds-price-dataset/input)  

* Dataset of Insomniac and normal people (классификация)  
Источник: [Mendeley Data](https://data.mendeley.com/datasets/jr5n4prgfv/1)  

---

## Метрики

* Для классификации используются две метрики: Accuracy и F1-score. Это позволяет оценить точность предсказаний и согласованность классификации между классами.

* Для регрессии выбраны три метрики: R², MAE и RMSE. Это позволяет оценить долю объяснённой дисперсии, среднюю величину ошибок и чувствительность модели к крупным отклонениям.
